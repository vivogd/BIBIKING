Example

There is an example scene called CardSwipeExampleScene that has the project setup for a reference on how this asset functions.


To get started

Create cards

I've created the cards as scriptable objects, which means they can be created from a menu item by right clicking in the project location, go to Create > Card > New Card

You may also create the cards programmatically if you prefer.

Give the Card a name (in the example project above they have been named after food) and then assign the card's Image field to a Sprite


Create your own Swipe Manager

You'll want to create your own Swipe Manager that is responsible for holding all of the cards, and handling if a card was liked, disliked, or if the user has finished the entire deck of cards. You can copy from the existing SwipeManager.cs that is provided.

In the example provided, the SwipeManager is attached to the parent Canvas. Cards are added to the allCards list.
